# SVM Kernel Analysis & PCA Visualization
Comparing different SVM kernels and using dimensionality reduction for complex classification tasks.

## Key Features
- **Dimensionality Reduction:** Implemented **Principal Component Analysis (PCA)** for 2D data visualization.
- **Kernel Comparison:** Evaluated **Linear vs. RBF kernels** and visualized their decision boundaries.
- **Evaluation:** Focused on AUC scores and boundary complexity.

## Technologies Used
- Scikit-learn (SVC, PCA), Matplotlib.
